Js中的css样式

Written by 陈晓拉尼 with ♥ on July 13, 2017 in JavaScript

js中css样式的不同 1、 JavaScript使用驼峰命名法，而css不是。 比如css中的‘border-radius’，js中的‘borderRadius’； 2、 JavaScript的属性值用字符串表示，而css不是。 比如js中div.style.backgroundColor='';css中div{}；
